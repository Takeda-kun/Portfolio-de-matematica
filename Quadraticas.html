Funções Quadráticas



As funções quadráticas são funções com equações de segundo grau, elas são formadas por três coeficientes, sendo eles o coeficiente “a”, o “b” e o “c”, esses coeficientes são muito importantes para toda a função, por que além de compô-la, eles irão mostrar alguns pontos da função no gráfico, onde irá cortar o eixo y, se ela é positiva ou negativa e, por que precisará fazer uma conta de Bháskara, e então precisar deles.


Exemplo de uma função quadrática:


f(x) = 1x2 + 2x + 3


Para encontrar esses coeficientes, é simples. O coeficiente “a” é acompanhado de um algarismo elevado ao quadrado, o coeficiente “b” é acompanhado por um outro algarismo, e, o coeficiente “c” não é acompanhado por nenhum algarismo.


Diferente de uma função de primeiro grau, que produzia uma reta no gráfico, a função quadrática produz uma reta curvada, conhecida como parábola, a parábola tem uma concavidade (a parte aberta dela), e, essa concavidade pode ser direcionada para cima ou para baixo, e isso irá depender se ela for negativa ou positiva, para saber isso, basta ver o coeficiente “a”, quando for negativo, terá a concavidade negativa (para cima) e quando for positivo, terá a concavidade positiva (para baixo).


Para encontrar as raízes ou o zero da função, também é simples. Tendo uma função quadrática, para achar as raízes, deve-se fazer uma conta com a fórmula de Bháskara, ela indicará quais serão os pontos de corte no eixo x do gráfico, mas, caso não tenha nenhuma raiz, significa que não tem corte no eixo x, para achar o 0 da função, basta substituir o “f(x)” por 0, então, saberá o valor do 0 da função.


Temos também algo chamado de Vértice, sendo ele o ponto da função no gráfico em quando a parábola começa a ter valores maiores, ou valores menores.

A fórmula para achar os vértices do x e do y são:


Para o x= -b / 2a

Para o y= - Δ / 4a


A função quadrática também tem outras formas e fórmulas, sendo elas:


Forma Geral: ax2 + bx + c = 0

Forma Canônica: a(x - xv)2 + yv = 0

Forma Fatorada: a(x - x1)(x – x2) = 0
